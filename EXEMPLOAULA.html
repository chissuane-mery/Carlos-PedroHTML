Seção de Parágrafos

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas da web. É uma linguagem de marcação que define a estrutura de um documento, usando elementos como parágrafos, tabelas e links.

CSS (Cascading Style Sheets) é usado para estilizar a aparência de páginas HTML. Com CSS, você pode controlar cores, fontes, espaçamentos e muito mais.
